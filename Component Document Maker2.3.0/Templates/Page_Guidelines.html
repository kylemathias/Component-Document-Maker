Editing Page Guidelines

 Making Changes:

  • Before you make any changes, please enable tracking in Microsoft Word. You can do this in the menu from Review -> Tracking and select "All Markup".  

 Adding New Components:

  • If you would like to add a New Component you can do so on the Quickwires page. First drag and drop the component you would like to create. Then click the "Component Document Maker" extension, then download. You will now have a 2nd document with the component you would like to add. You can now copy the component and paste it into this page document.  

 Saving This Document:

  • If you have any issues saving this file, save this document as a new file. You can do so in the menu with File -> Save As
