Editing Page Guidelines

 Making Updates to this page:

 Please highlight your updates using this system:

  • Initial requested changes: Highlighted in Green  
  • After review, second round of updates: Highlighted in Yellow  
  • To remove copy, use the strikethrough tool and highlight. Example: Second Round Remove  

 Adding New Components:

  • If you would like to add a New Component you can do so on the Quickwires page. First drag and drop the component you would like to create. Then click the "Component Document Maker" extension, then download. You will now have a 2nd document with the component you would like to add. You can now copy the component and paste it into this page document.  

 Saving and Sharing this Document:

  • Enable AutoSave in Microsoft Word (top left in the document window)
  • Click Share in Microsoft Word (top right in the document window)  
  • Change "People you specify can edit" to "People in NetApp Inc. with the link"  
  • Then click "Copy" under "Copy link"
