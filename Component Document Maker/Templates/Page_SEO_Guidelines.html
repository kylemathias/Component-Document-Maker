SEO Engagement Guidelines

Engage with the SEO team for keyword suggestions if your changes fall under the following criteria.

Major Updates:

The SEO team will have insight on previous page keyword performance and possible industry updates that might help optimize page performance when updating the following:

  • The Hero (new title, new subhead, new body)
  • Any SEO Component content changes (URL path, title, description)
  • Page Component Metadata

Creating a New Page:

It's best to work with the SEO team to create an SEO strategy before the initial page design is complete. The SEO strategy includes the keyword(s) for the following:

  • Hero (title, subhead, body)
  • SEO Component content (URL path, title, description)
  • Page Component Metadata
