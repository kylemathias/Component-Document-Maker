Guidelines: 

Content Image/Video (optional)- max-height 350px

Content Headline (required)- up to 60 characters, plain text. Will be page title if in hero region.

Content Body (required) - up to 600 characters, rich text

Content CTA (optional) - not recommended unless an asset download

Second Content Image (optional)- transparent, graphical .png with two exposed sides, 4:3 aspect ratio

 

Form headline (optional, but recommended) up to 60 characters, plain text

Form ID (required) - Provide Marketo Form ID

Form description (optional) - instructions, up to 200 characters, plain text.

Form declaration (optional) - legal text, up to 200 characters, plain text. 

Form submit button label (required)

 

Confirmation/Thanks Headline (required) - Confirm/thank headline, up to 60 characters, plain text. 

Confirmation/Thanks Message (required) - Confirm/thank message, up to 200 characters, plain text.
